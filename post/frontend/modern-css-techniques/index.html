现代 CSS 技巧：Glassmorphism 与 CSS 变量

现代 CSS 已经强大到令人惊讶的程度，无需 JavaScript 就能实现很多复杂的视觉效果。本文总结几个实用的现代 CSS 技巧。

CSS 自定义属性（变量）

CSS 变量是构建设计系统的基础：

:root {
  --color-primary: #6366f1;
  --color-bg: #0f0f1a;
  --color-surface: rgba(255, 255, 255, 0.05);
  --border-radius: 16px;
  --transition: 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* 亮色主题覆盖 */
[data-theme="light"] {
  --color-bg: #f8fafc;
  --color-surface: rgba(0, 0, 0, 0.04);
}

主题切换只需在 html 元素上切换 data-theme 属性，所有颜色自动更新！

Glassmorphism 效果

毛玻璃效果的核心是 backdrop-filter：

.glass-card {
  background: rgba(255, 255, 255, 0.08);
  backdrop-filter: blur(20px) saturate(180%);
  -webkit-backdrop-filter: blur(20px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.12);
  border-radius: 16px;
  box-shadow: 
    0 8px 32px rgba(0, 0, 0, 0.3),
    inset 0 1px 0 rgba(255, 255, 255, 0.1);
}

关键参数：

  • blur() 值越大，背景越模糊（通常 12px~24px）
  • background 透明度控制玻璃的“厚度“
  • border 用半透明白色模拟玻璃边缘折射

流畅动画

.card {
  transform: translateY(0);
  transition: transform var(--transition), box-shadow var(--transition);
}

.card:hover {
  transform: translateY(-8px);
  box-shadow: 0 20px 60px rgba(99, 102, 241, 0.3);
}

用 cubic-bezier 替代线性动画，感觉更自然。

容器查询（Container Queries）

比媒体查询更精准：

.card-container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card {
    flex-direction: row;
  }
}

CSS Grid 高级布局

.posts-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(320px, 1fr));
  gap: 24px;
}

auto-fill + minmax 实现自适应列数，无需媒体查询！

渐变文字

.gradient-text {
  background: linear-gradient(135deg, #6366f1, #8b5cf6, #06b6d4);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}

总结

现代 CSS 的能力远超 10 年前的想象：

  • CSS 变量：设计系统和主题切换的基础
  • backdrop-filter：毛玻璃效果
  • 容器查询：组件级响应式
  • CSS Grid：强大的二维布局

拥抱这些特性，你会发现大量 JavaScript 操控样式的代码可以用纯 CSS 替代。
